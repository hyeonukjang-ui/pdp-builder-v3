# Phase 3: Quick Win A/B 테스트

> 오너: 박윤경(설계) + 개발팀(구현) | 기간: 3~4주 (5월 4주차 ~ 6월 3주차) | 상태: 대기

---

## 목표

AI로 생성한 PDP가 기존 대비 CVR을 실제로 올리는지 검증. 가장 빠르게 임팩트를 증명할 수 있는 2개 실험 실행.

## 실험 설계

### 실험 A: AI 생성 PDP vs 기존 PDP

| 항목 | 내용 |
|------|------|
| **대조군** | 현행 PDP (콘텐츠 부실 상태 그대로) |
| **실험군** | Phase 2에서 AI 생성한 이미지+카피가 적용된 PDP |
| **대상** | Phase 2 적용 30개 상품 |
| **주요 지표** | PDP → 예약 전환율 (CVR) |
| **보조 지표** | CTA 클릭률(CTR), 이탈률, 평균 체류 시간, 스크롤 깊이 |
| **예상 기간** | 표본 크기 산정 후 확정 (최소 2주) |
| **난이도** | 중간 — PDP 콘텐츠 교체 |

### 실험 B: CTA 문구 변경

| 항목 | 내용 |
|------|------|
| **대조군** | "예약하기" (현행) |
| **실험군** | "날짜 확인하기" (투어), "가격 보기" (티켓) |
| **대상** | Phase 2 적용 30개 상품 |
| **주요 지표** | CTA 클릭률 (CTR), 예약 전환율 (CVR) |
| **보조 지표** | 이탈률, 평균 체류 시간 |
| **예상 기간** | 표본 크기 산정 후 확정 (최소 2주) |
| **난이도** | **낮음** — 텍스트 변경만 |

### 실험 C: 무료취소 배지 위치 (선택)

| 항목 | 내용 |
|------|------|
| **대조군** | 취소정책 페이지 하단 (현행) |
| **실험군** | trustBadges로 ATF 상단 배치 |
| **대상** | 무료취소 가능 상품 중 Phase 2 적용분 |
| **주요 지표** | CVR, CTA CTR |
| **보조 지표** | 취소정책 섹션 도달률 변화 |
| **난이도** | **낮음** — 위치 이동 |

## 플랜 B: A/B 인프라 부재 시

| 대안 | 방법 | 한계 |
|------|------|------|
| Before/After 시계열 분석 | 적용 전/후 CVR 변화 + 베이지안 추론 | 외부 변수 통제 어려움 |
| 대상 상품군 확대 | 30개 → 50개로 늘려 통계적 파워 확보 | 생성 비용 증가 |
| 매칭 비교군 | 유사 트래픽/카테고리 상품과 비교 | 완벽한 통제 불가 |

### Go/No-Go 판단 기준

| 결과 | 판단 | 다음 액션 |
|------|------|----------|
| CVR +1pp 이상, p<0.05 | **Go** | Phase 4~5 진행 |
| CVR +0.5~1pp, p<0.1 | **Conditional Go** | 대상 확대 후 재실험 |
| CVR 변화 없음 또는 하락 | **Pivot** | 생성 콘텐츠 품질 재검토 + 유저 인터뷰 |
