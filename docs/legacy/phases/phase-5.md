# Phase 5: 스케일링 — 257개 전체 적용

> 오너: 기원(자동 생성) + 현욱(빌더 일괄 적용) + 박윤경(품질 모니터링) | 기간: 4~6주 (8~9월) | 상태: 대기

---

## 목표

Phase 2~4에서 검증된 AI 콘텐츠 생성 패턴을 257개 콘텐츠 임계점 미달 상품 전체에 확대 적용. Grade D → C 이상 전환.

---

## 스케일링 전략

```
Phase 2: 30개 상품 (수동 검수 100%)
Phase 5: 257개 상품 (자동 생성 + 샘플 검수)

핵심 변화:
- 수동 검수 → 자동 품질 검수 파이프라인
- 개별 적용 → 일괄 생성 + 일괄 적용
- 결과 확인 → 자동 모니터링 대시보드
```

---

## 일괄 생성 프로세스

```
[1] 257개 상품 리스트 → 카테고리별 분류
    ↓
[2] 카테고리별 전환 레시피 자동 매핑
    ↓
[3] AI 카피 생성 (배치 처리, 병렬)
    ├─ 성공 → [4]로
    └─ 실패 → 재시도 큐 (최대 3회)
    ↓
[4] AI 이미지 생성 (배치 처리)
    ↓
[5] 자동 품질 검수
    ├─ Pass → [6]으로
    └─ Fail → 수동 검수 큐
    ↓
[6] 빌더 일괄 적용
    ↓
[7] Health Score 재산출 + CVR 모니터링 시작
```

### 비용 추정 (257개)

| 항목 | 단가 | 257개 합계 |
|------|------|----------|
| AI 카피 생성 | ~$0.03/상품 | ~$7.71 |
| AI 이미지 생성 | ~$0.2~1.0/상품 (5~8장 기준) | ~$50~250 |
| 총 API 비용 | | **~$60~260** |
| 소요 시간 (병렬 처리) | ~30초/상품 (카피) | ~2시간 (카피만) |

> 이미지 비용은 DALL-E 기준 $0.04~0.12/장. 생성 모델/해상도에 따라 변동.

## 품질 모니터링 프레임워크

### 자동 검수 기준

| 항목 | 기준 | 실패 시 액션 |
|------|------|------------|
| 카피 길이 | highlights 5개, overview 100~300자 | 수동 검수 큐 |
| 필수 필드 존재 | 카테고리별 필수 블록 100% | 재생성 |
| 금지 필드 분리 | 가격/취소정책/좌표 = AI 생성 아님 | 자동 수정 |
| 팩트 체크 | 원본 데이터와 명백한 모순 없음 | 수동 검수 큐 |
| 이미지 해상도 | 최소 1200px 너비 | 재생성 |

### CVR 모니터링

```
주간 체크:
- 적용 상품 CVR 변화 (Before/After)
- Grade 분포 변화 (D→C 전환율)
- Health Score 평균 변화

경보 조건:
- 적용 상품 CVR이 적용 전 대비 -5% 이상 하락 → 자동 알림
- Grade D 잔존 상품 > 20% → 원인 분석
```
