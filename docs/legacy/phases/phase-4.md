# Phase 4: 리뷰/UGC 강화

> 오너: 박윤경(설계) + 기원(AI 리뷰 요약) | 기간: 2~3주 (7월) | 상태: 대기 (Phase 3 Go 후)

---

## 목표

전환 영향 1위 레버(리뷰/UGC)를 AI 생성 PDP에 추가. 리뷰가 있는 상품의 CVR을 베이스라인 대비 +10% 끌어올린다.

---

## 왜 리뷰/UGC가 Phase 4인가

```
리서치 결과: 리뷰 有 vs 無 = +354% CVR (방향성 근거, PowerReviews)
→ 가장 강력한 레버지만, 리뷰 데이터 API 접근이 필요해 Phase 2보다 늦게 시작

Phase 2~3에서 먼저 "콘텐츠 존재 여부"의 CVR 영향을 검증하고,
Phase 4에서 "리뷰/UGC 추가"의 추가 CVR 영향을 검증하면
→ 각 레버의 개별 기여도를 분리 측정 가능
```

---

## 태스크 체크리스트

### 리뷰 데이터 통합
- [ ] **4-1** 리뷰 데이터 API 접근 권한 확보 + 데이터 구조 파악
- [ ] **4-2** 상품별 리뷰 수/평점 데이터 수집
- [ ] **4-3** 리뷰 별점 ATF 배치 구현 (hero 블록 내 또는 바로 아래)

### AI 리뷰 요약 (선택)
- [ ] **4-4** 리뷰 요약 프롬프트 설계 (긍정/부정/팁 3카테고리)
- [ ] **4-5** 리뷰 30개 이상 상품 대상 AI 요약 생성
- [ ] **4-6** 요약 품질 검수 (원본 리뷰와 대조)

### UGC 갤러리 (선택)
- [ ] **4-7** UGC 이미지 수집 가능 여부 확인
- [ ] **4-8** UGC 갤러리 블록 설계 + 빌더 추가

### 검증
- [ ] **4-9** 리뷰 有 상품 CVR 변화 측정 (Before/After 또는 A/B)

## 리뷰 배치 전략

### ATF 리뷰 요약

```
┌─────────────────────────────────┐
│  hero (대표 이미지 + 제목)       │
│  ★★★★☆ 4.7 (328개 리뷰)       │  ← 리뷰 별점 ATF 배치
│  ✓ 무료취소  ✓ 즉시확정         │
│  [날짜 확인하기]                 │
└─────────────────────────────────┘
```

### AI 리뷰 요약 블록

```
┌─────────────────────────────────┐
│  📝 328개 리뷰 요약              │
│                                  │
│  ✅ 좋은 점: "가이드 설명이..."   │
│  ⚠️ 참고: "비가 오면 일정이..."  │
│  💡 팁: "아침 일찍 가면..."      │
└─────────────────────────────────┘
```
