# Phase 0: 방향 합의 + ROI 추정

> 오너: 박윤경 | 기간: 1주 (3월 4주차) | 상태: **진행중**

---

## 목표

TF 내부 방향성 합의. "상품소개가 없는 상품에 AI로 이미지+카피를 생성한다"는 핵심 프레이밍 확립 + ROI 추정 공유.

---

## 핵심 프레이밍

```
현재: 659개 상품 중 257개가 콘텐츠 임계점 미달. 118개는 통이미지 PDP (SEO 0점, 접근성 0점).
문제: 수동 제작 시 PDP 1건 = 2~5일. 257개 × 3.5일 = 900일 = 비현실적.
해결: AI로 이미지 + 카피를 자동 생성하되, 전환율 높은 구조로 생성한다.
```

### TF 역할 재정의

| 멤버 | 역할 | 이 프로젝트에서 풀고 있는 문제 |
|------|------|------|
| 기원 | AI 콘텐츠 생성 | 상품 데이터 → AI → 이미지+카피 자동 생성 파이프라인 |
| 현욱 | 섹션 빌더 | 생성된 콘텐츠를 PDP에 조립하는 도구 |
| 박윤경 | 전환 설계 | 어떤 상품 먼저, 어떤 블록 구조로, 어떤 톤으로 생성할지 설계 |

### ROI 추정 공식

```
예상 추가 GMV = Σ (상위 30개 상품의 일평균 PDP뷰 × 현재 CVR × 예상 CVR 상승률 × 평균 객단가 × 30일)

예상 CVR 상승률: 보수적 +1~2pp (Phase 3 A/B 결과로 정밀화)
데이터 소스: BigQuery (일매출, PDP뷰, CVR 이미 존재)
```
