# 전문가 심층 비평

대상: $ARGUMENTS

## 실행 절차

1. **대상 파악**: 위에 지정된 문서/코드/설계를 읽는다.

2. **전문가 1인 동적 생성**: 대상에 가장 적합한 전문가 1인을 설계한다.
   - 해당 분야의 **시니어급** (8~15년차) 페르소나
   - **비판적이되 건설적인** 성향
   - 리뷰 렌즈 5~7개로 깊이 있는 분석

3. **심층 비평 실행**: Task 도구로 에이전트를 스폰하여 심층 비평을 수행한다.

4. **출력 형식**:

```
## 전문가 프로필
{직무, 경력, 전문분야}

## 총평 (3~5문장)

## 심층 분석

### 강점 (유지해야 할 것)
1. {구체적 강점 + 왜 좋은지}

### 약점 (개선해야 할 것)
1. {문제} → {근거} → {구체적 액션}

### 놓친 관점 (고려하지 못한 것)
1. {빠진 요소 + 왜 중요한지 + 어떻게 보완할지}

### 실행 우선순위
1. (즉시) ...
2. (1주 내) ...
3. (장기) ...
```
